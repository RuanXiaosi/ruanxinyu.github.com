每天学习一个Linux命令（2）：cd命令

2018-05-01
  • Linux

cd命令用于切换至目标目录，以/开头的则为绝对路径，否则为相对路径。

语法

使用小技巧

  1. 切换至用户所示在的目录可以使用cd ~ 或者 cd(不带参数)
  2. "." 则是表示目前所在的目录，".." 则表示目前目录位置的上一层目录
  3. 通过pwd命令可以打印当前所在的目录

常用范例

范例1： 切换至根目录下的var目录下
命令： cd /var(绝对路径)

范例2： 切换至当前目录下的var目录下
命令： cd var(相对目录)

范例3： 切换至上两级的目录
命令： cd ../../

范例4： 切换至包含空格的目录
命令： cd 'aa aa'或者cd aa\ aa

范例5： 切换至当前用户的home目录
命令： cd ~或者cd

范例5： 打印当前所在的目录
命令： pwd
